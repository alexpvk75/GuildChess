# GuildChess

Il progetto si propone di modernizzare la gestione di tornei scacchistici. GuildChess, come una piattaforma desktop, integra:
- Un motore di abbinamento che supporta girone all'italiana, eliminazione, sistema svizzero e Scheveningen
- Un modulo di analisi che valuta le partite in termini di precisione tattica, efficienza strategica e gestione del tempo
- Strumenti per la gestione di circolo e gilde, con monitoraggio dei progressi individuali, coordinamento di tornei e modulo di pianificazione 
- Visualizzazioni grafiche di classifiche e tabelloni
